# 消费分类指南

本文档总结了账单流水的分类方法，用于指导 AI 或人工对交易记录进行分类。

## 分类类目

| 类目 | 说明 |
|------|------|
| 吃饭 | 餐饮、食品、饮料、超市食品采购 |
| 出行 | 出租车、网约车、共享单车、地铁、公交等交通出行 |
| 旅游 | 景点门票、住宿、火车票、机票、旅行相关消费、旅途中的转账分摊 |
| 工作 | 云服务、平台认证、工作相关转账、劳动报酬 |
| 服饰化妆 | 衣服、化妆品、首饰、美甲、造型等让自己看起来更好看得体的消费 |
| 手工爱好 | 编织、毛线、布料、面料、缝纫工具、染料、刺绣材料等手工DIY相关 |
| 娱乐 | 玩偶、手办、预售商品、电影票、演出等娱乐消费 |
| 慈善 | 捐款、公益筹款、寄送物资 |
| 日用百货 | 拼多多、小红书等平台的日用品购物 |
| 转账 | 余额宝收益、信用卡还款、花呗还款、基金交易等资金流转 |
| 其他 | 红包、无法归类的消费、理发等个人服务 |

## 分类规则

### 1. 按交易对方关键词判断

**吃饭**
- 餐饮店、餐厅、面馆、小吃店、快餐店
- 关键词：餐饮、美食、面、饭、炸鸡、烧烤、火锅、麻辣烫、奶茶、糖水
- 商家示例：麦当劳、肯德基、星巴克、蜜雪冰城、瑞幸咖啡
- 山姆/沃尔玛购买食品类（备注说明为"买吃的"）

**出行**
- 关键词：打车、专车、快车、先乘后付、骑行、地铁、公交
- 商家示例：滴滴出行、花小猪打车、高德打车、曹操出行
- 包含：共享单车、骑行等短途出行
- **注意**：美团需看明细，"先骑后付"是出行，但餐饮店消费是吃饭

**旅游**
- 关键词：景点、寺庙、滑雪场、旅行、民宿、酒店、火车票、机票
- 商家示例：同程旅行、携程、飞猪、铁路12306、各类景区
- **重要**：火车票、机票都属于旅游，不是出行
- 特殊：旅途中给朋友的转账（需结合场景判断）

**手工爱好**
- 关键词：编织、毛线、布料、面料、缝纫、裁剪、刺绣、染色、染料、织布、手工DIY
- 商家示例：手工材料店、布料店、毛线店
- 包含：编织机、缝纫剪、织布机等工具

**工作**
- 关键词：云服务、认证、企业、小程序、VIP（工作类）
- 商家示例：腾讯云、阿里云、AWS、微信认证
- 特殊：给同事/财务的工作相关转账

**服饰化妆**
- 所有买衣服的、化妆品、首饰，和让自己看起来更好看得体的内容全部归属于这个类目
- 关键词：服装、服饰、衣、裤、鞋、首饰、饰品、美甲、造型、化妆品、护肤、口红、面膜
- 商家示例：优衣库、ZARA、丝芙兰、屈臣氏
- **注意**：布料、面料属于手工爱好，不是服饰化妆

**娱乐**
- 关键词：预售、玩偶、手办、电影、演出
- 商家示例：淘票票、猫眼电影、大麦网

**慈善**
- 关键词：筹款、捐赠、公益、爱心
- 商家示例：水滴筹、轻松筹、腾讯公益

**日用百货**
- 关键词：日用、百货、超市
- 商家示例：拼多多、小红书（非服饰类）、盒马、奥乐齐

**转账**
- 交易对方为基金公司且明细包含"余额宝"和"收益发放" → 转账
- 信用卡还款 → 转账
- 花呗还款（花呗主动还款、花呗账单等）→ 转账
- 基金交易（基金买入、基金卖出、ETF等）→ 转账
- 余额宝转出到银行卡 → 转账

### 2. 美团交易的特殊处理

美团作为平台，需要根据明细判断实际消费类型：

| 明细关键词 | 分类 |
|-----------|------|
| 先骑后付 | 出行 |
| 餐厅名、糖水、奶茶等食物 | 吃饭 |
| 景点名、滑雪场 | 旅游 |
| 酒店、民宿 | 旅游 |

### 3. 按交易类型判断

**微信红包**
- 普通红包收发 → 其他
- 特定场景红包（如请客吃饭发红包）→ 吃饭

**转账**
- 需结合转账备注和上下文判断
- 朋友间旅游分摊 → 旅游
- 工作相关 → 工作
- 请客吃饭 → 吃饭

**群收款**
- 通常为聚餐分摊 → 吃饭

### 4. 特殊情况处理

1. **退款**：与原消费保持相同分类
2. **小红书**：需看具体商品
   - 购买服饰 → 服装首饰
   - 工作用途 → 工作
   - 日用品 → 日用百货
3. **山姆/超市**：需看具体购买内容
   - 食品 → 吃饭
   - 日用品 → 日用百货
4. **二维码付款给个人**：需结合场景判断
   - 路边摊吃饭 → 吃饭
   - 旅途中消费 → 旅游
   - 无法判断 → 其他

### 5. 无法判断时

- 优先标记为"其他"
- 可在备注说明中记录疑问，后续人工复核
